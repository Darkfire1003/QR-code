### What I learned

While building this project, I focused on understanding layout and positioning using pure CSS instead of relying on frameworks.

Key learnings:

- Centering elements using Flexbox (`justify-content` and `align-items`)
- Using `max-width` to control responsive layouts
- Structuring clean and semantic HTML
- Improving visual accuracy by adjusting spacing, font sizes, and colors

Example of centering the layout:

```css
body {
  min-height: 100vh;
  display: flex;
  flex-direction: column;
}

main {
  flex: 1;
  display: flex;
  justify-content: center;
  align-items: center;
}
```
